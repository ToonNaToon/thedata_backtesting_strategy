# 0DTE Iron Condor Backtesting Strategy

A comprehensive backtesting framework for 0-day-to-expiration (0DTE) iron condor strategies on SPXW, SPY, QQQ, and SPX.

## Features

- **Flexible Parameters**: Configure ticker, wing sizes, entry/exit times, and day exclusions
- **Interactive Dashboard**: Streamlit-based web interface for parameter selection and results visualization
- **Multiple Entry Times**: Test single or multiple entry times simultaneously
- **Day of Week Analysis**: Analyze performance by trading day
- **Comprehensive Statistics**: Win rate, P&L metrics, and detailed trade results
